## üí¨ AWS Support Plans

AWS offers multiple **support tiers** designed to match different customer needs.

| Support Plan   | Key Features                                                                                         | Response Times                                             |
| -------------- | ---------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| **Developer**  | Business-hour email support; access to Cloud Support Associates.                                     | General guidance: < 24 hrs System impaired: < 12 hrs    |
| **Business**   | 24/7 phone, email, and chat; access to Cloud Support Engineers.                                      | System impaired: < 4 hrs Production system down: < 1 hr |
| **Enterprise** | Dedicated **Technical Account Manager (TAM)** and **Concierge Team** for billing and best practices. | Business-critical system down: < 15 mins                   |

> üß† **Exam Tip:** Remember the support response times and the role of the **TAM** in Enterprise Support.



## üõí AWS Marketplace

A **digital catalog** of thousands of software listings from **independent vendors** (third parties).

| Feature                | Description                                                                                |
| ---------------------- | ------------------------------------------------------------------------------------------ |
| **Buy Solutions**      | Purchase AMIs, SaaS, container products, or CloudFormation templates ready for production. |
| **Integrated Billing** | All purchases are billed through your **AWS account** ‚Äî no separate vendor billing.        |
| **Sell Solutions**     | You can list and sell your own products through the AWS Marketplace.                       |

> üí° Example: Purchase a preconfigured security firewall AMI directly through AWS Marketplace.

## üßë‚Äçüíº AWS Professional Services & Partner Network (APN)

AWS partners with global experts and consulting firms to help customers build and optimize AWS environments.

| Partner Type                  | Description                                                                                      |
| ----------------------------- | ------------------------------------------------------------------------------------------------ |
| **AWS Professional Services** | A team of AWS experts who collaborate directly with your team on implementation.                 |
| **APN Technology Partners**   | Provide **hardware**, **connectivity**, and **software** that integrate with AWS infrastructure. |
| **APN Consulting Partners**   | Professional service firms that help you **design, build, and manage** AWS solutions.            |
| **APN Training Partners**     | Authorized organizations that deliver official AWS training programs.                            |
| **Competency Program**        | Recognition for APN partners who have proven expertise in specialized solution areas.            |
| **Navigate Program**          | Helps APN partners improve their skills, specialization, and service delivery.                   |
