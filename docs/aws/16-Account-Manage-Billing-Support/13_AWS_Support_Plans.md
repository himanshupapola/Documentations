# üí∞ AWS Support Plans Overview

AWS offers **four tiers of Support Plans**, each designed to meet different levels of business needs ‚Äî from free community support to enterprise-grade 24/7 assistance.  
Knowing these differences is **important for the AWS Certified Cloud Practitioner exam**.

| Support Plan                        | Intended For                | Monthly Cost              | Key Access                            |
| ----------------------------------- | --------------------------- | ------------------------- | ------------------------------------- |
| **Basic**                           | All AWS accounts (default)  | ‚úÖ Free                   | Self-service resources                |
