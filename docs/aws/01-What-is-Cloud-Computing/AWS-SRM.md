AWS and the customer share responsibilities for security and compliance in the cloud.

---
## AWS Responsibilities (Security of the Cloud)

- Protects the physical infrastructure, hardware, and software that run AWS services.
- Manages networking, storage, and virtualization security.
- Ensures compliance of underlying cloud infrastructure.
- Infrastructure (global network security)
- Configuration and vulnerability analysis
- Compliance validation
---
## Customer Responsibilities (Security in the Cloud)

- Configure and manage data, applications, and access controls.
- Protect operating systems, platforms, and network traffic within your instances.
- Ensure compliance for workloads deployed on AWS services.
